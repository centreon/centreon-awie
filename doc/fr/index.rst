Bienvenue dans la documentation du module Centreon AWIE !
=========================================================

Le module Centreon AWIE a été conçu pour aider les utilisateurs à configurer plusieurs plateformes Centreon Web, d'une manière plus rapide et plus facile, grâce à son mécanisme d'import/export.

Il peut se révéler utile pour configurer un environnement de pré-production afin de vérifier si toute la configuration est correcte avant d'exporter celle-ci vers l'environnement final.

Centreon AWIE s'appuie sur les commandes CLAPI mais possède l'avantage de pouvoir les exécuter depuis l'interface graphique au lieu d'avoir à les taper dans un terminal.
